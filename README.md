## 🖌 Overview  

iZotope Nectar Professional Suite is a comprehensive vocal production environment that integrates with iZotope's broader ecosystem to share analysis data, sidechaining, and processing intelligence across multiple tools in a connected session.

MeloToolkit within Nectar Professional Suite provides MIDI-controlled pitch shifting and vocal effect capabilities that expand creative vocal processing beyond correction into performance transformation and experimental vocal sound design.

Inter-plugin communication with iZotope Relay and other ecosystem tools allows Nectar to respond intelligently to the content of other tracks in the session, adjusting vocal processing based on the presence and level of competing instruments automatically.

Detailed module control exposes every parameter of every processing stage for users who want to go beyond the intelligent starting points and apply precise manual settings calibrated to the specific characteristics of the vocal being processed.

iZotope Nectar Professional Suite is the vocal processing platform for engineers and producers who want intelligent automation for speed and manual depth for precision, combined in a plugin that treats the vocal as the center of the mix it deserves to be.
